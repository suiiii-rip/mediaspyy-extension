# MediaSpyy

A browser extension that listens to the MediaSession API and publishes played
media titles and URLs to a configured backend.

## Setup

```bash
npm install
```

## Build

```bash
npm run build
```

Watch mode:

```bash
npm run watch
```

## Loading extension

Load the `dist` folder as an unpacked extension.
